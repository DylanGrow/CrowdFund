How does the Content Security Policy protect my browser?

Our strict HTTP Content Security Policy (CSP) header instructs your browser to refuse any JavaScript execution that does not originate from our signed, compiled static files. It blocks inline injections, remote script loading, framing (anti-clickjacking), and external font API fetches, making this site extremely secure against cross-site scripting (XSS).
